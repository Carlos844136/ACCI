# ACCI-A

Site estático para a campanha ACCI-A, Aliança contra a Caça Ilegal na Amazônia.

## Como abrir

Abra `view/index.html` diretamente no navegador ou sirva a pasta `view` com um servidor local:

```bash
cd view
python -m http.server 4173
```

Depois acesse `http://127.0.0.1:4173/`.

## Arquivos principais

- `view/index.html`: estrutura da página
- `view/styles.css`: layout, responsividade e identidade visual
- `view/script.js`: menu mobile, modal Pix, denúncia anônima/identificada, máscara de CPF, upload demonstrativo e envio do formulário
- `imgs/`: imagens de apoio e previews
